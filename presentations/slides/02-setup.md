    <div style="background:#fff;border-radius:6px;padding:0.55rem 0.7rem;">

# Or API key directly:
export ANTHROPIC_API_KEY=sk-ant-...</code></pre>

    </div>

  </div>

          Frontend: React 19, TypeScript, Vite, TanStack Query, Recharts
          Database: MySQL 8.0 — double-entry accounting schema
          DevOps:   GitHub Actions CI/CD → RobiCloud (Docker)

      </div>

      </p>
    </div>

    <div class="feature-card">
      <div style="display:flex;align-items:center;gap:0.5rem;margin-bottom:0.5rem;">

        <span  class="card-title-t">Specific — Claude delivers first time</span>
      </div>
      <div class="prompt-box" style="font-size:0.58em;">
        Create a Spring Boot @Service class AccountingService.java in
        com.wamy.finance.service. Use constructor injection. Method:
        postJournalEntry(JournalEntryRequest dto) — validates debit == credit,
        saves to journal_entries via JPA, publishes a JournalEntryPostedEvent.
        Return JournalEntryResponseDTO.
      </div>
      <p  class="card-body-t">
        ✓ Specific: package · pattern · method signature · validation · return type
      </p>
    </div>

 <h1 class="slide-title">Claude Code Essential Commands</h1>
  <p class="slide-title=text">The 10 commands every developer must know</p>



  <div class="cap-pill-t-l">
    <code style="background:#24337F;color:#9FBFFF;font-size:0.8em;padding:0.2em 0.6em;border-radius:3px;white-space:nowrap;font-weight:700;">/init</code>&nbsp;&nbsp;
    Scan project and generate CLAUDE.md context file. Run once per project.
  </div>
  
  <div class="cap-pill-t-l">
    <code style="background:#24337F;color:#9FBFFF;font-size:0.8em;padding:0.2em 0.6em;border-radius:3px;white-space:nowrap;font-weight:700;">/run &lt;cmd&gt;</code>&nbsp;&nbsp;
    Run a shell command. Claude sees the output and can react to errors.
  </div>





   <div class="cap-pill-t-l">
    <code style="background:#24337F;color:#9FBFFF;font-size:0.8em;padding:0.2em 0.6em;border-radius:3px;white-space:nowrap;font-weight:700;">/add &lt;file&gt;</code>&nbsp;&nbsp;
    Add file(s) to Claude's context window. Critical for multi-file tasks.
  </div>

  <div class="cap-pill-t-l">
    <code style="background:#24337F;color:#9FBFFF;font-size:0.8em;padding:0.2em 0.6em;border-radius:3px;white-space:nowrap;font-weight:700;">/test</code>&nbsp;&nbsp;
    Run your test suite. Claude reads failures and proposes fixes.
  </div>

   <div class="cap-pill-t-l">
    <code style="background:#24337F;color:#9FBFFF;font-size:0.8em;padding:0.2em 0.6em;border-radius:3px;white-space:nowrap;font-weight:700;">/clear</code>&nbsp;&nbsp;
    Clear conversation history. Start fresh while keeping CLAUDE.md.
  </div>


<div class="cap-pill-t-l">
    <code style="background:#24337F;color:#9FBFFF;font-size:0.8em;padding:0.2em 0.6em;border-radius:3px;white-space:nowrap;font-weight:700;">/commit</code>&nbsp;&nbsp;
    Stage and commit all Claude changes with an AI-generated commit message
  </div>
       

    <div class="cap-pill-t-l">
      <code style="background:#24337F;color:#9FBFFF;font-size:0.8em;padding:0.2em 0.6em;border-radius:3px;white-space:nowrap;font-weight:700;">/undo</code>
      Undo the last file change Claude made. Safe rollback.
    </div>

    <div class="cap-pill-t-l">
      <code style="background:#24337F;color:#9FBFFF;font-size:0.8em;padding:0.2em 0.6em;border-radius:3px;white-space:nowrap;font-weight:700;">/search &lt;term&gt;</code>
      Search codebase for term. Claude uses results as context.
    </div>

    <div class="cap-pill-t-l">
      <code style="background:#24337F;color:#9FBFFF;font-size:0.6em;padding:0.2em 0.6em;border-radius:3px;white-space:nowrap;font-weight:700;">/diff</code>
      Show all pending changes Claude has made but not yet saved.
    </div>

    <div class="cap-pill-t-l">
      <code class="code-block-t">/help</code>
      List all available commands. Always up to date.
    </div>
