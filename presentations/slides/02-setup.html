Claude Code — Installation & Authentication

3 steps to your first AI-assisted commit

Prerequisites

node --version
java --version
mvn --version
# Need Node.js 18+
# Need Java 21 for WAMY
# Maven 3.9+

Install Claude Code

npm install -g @anthropic-ai/claude-code

# Verify
claude --version

Authenticate

claude login
# Opens browser → Anthropic OAuth

# Or API key directly:
export ANTHROPIC_API_KEY=sk-ant-...

Launch on WAMY project

cd wamy-finance
claude
# Claude scans project and is ready
💡 For WAMY: create a wamy-finance/ root folder containing both the Spring Boot backend and React frontend before running claude.

# WAMY NGO — Finance Module

## Project
SaaS monolithic ERP | Finance module | Laravel → Spring Boot 3.2 + React 19

Backend:  Spring Boot 3.2, Java 21, Maven, Spring Security (JWT RS256)

REST API: /api/v1/** with versioning
Auth:     JWT Bearer tokens
Roles:    ADMIN | FINANCE_MANAGER | ACCOUNTANT | AUDITOR

          chart_of_accounts, projects

Rule:   Every POST to journal_entries requires debit_total == credit_total

IDE Integration — VS Code + Claude Code

Inline suggestions · Side panel · File context · Diff review

✏️ Inline Edit Select code → Ctrl+K → describe change → Claude edits inline with diff preview
💬 Side Panel Ctrl+Shift+C → full conversation with code context always available alongside your editor
📁 File Context @filename in chat adds a file to context; /add to bulk-add entire folders
🔍 Diff Review Every change shown as a diff — accept or reject line by line, or accept all at once
⚡ Quick Fix Red squiggle → "Fix with Claude" → Claude reads the compiler error and proposes a fix
🧪 Test Runner "Run tests with Claude" → Claude reads failures and proposes fixes automatically until tests pass
💡 Install both the "Claude Code" VS Code extension AND use the terminal — the terminal is more powerful for multi-file tasks.

Prompt Patterns That Work

How to write prompts that get great results every time

BAD Vague — Claude has to guess
Write me a Java class

⚠ Too vague — Claude guesses your stack, your package, your pattern, and your purpose

GOOD Specific — Claude delivers first time
Create a Spring Boot @Service class AccountingService.java in com.wamy.finance.service. Use constructor injection. Method: postJournalEntry(JournalEntryRequest dto) — validates debit == credit, saves to journal_entries via JPA, publishes a JournalEntryPostedEvent. Return JournalEntryResponseDTO.

✓ Specific: package · pattern · method signature · validation · return type

The 5 Elements of a Great Prompt:
1. File / Class name 2. Package / folder 3. Framework / pattern 4. Method signature 5. Expected behaviour

Claude Code Essential Commands

The 10 commands every developer must know

/init Scan project and generate CLAUDE.md context file. Run once per project.
/run <cmd> Run a shell command. Claude sees the output and can react to errors.
/add <file> Add file(s) to Claude's context window. Critical for multi-file tasks.
/test Run your test suite. Claude reads failures and proposes fixes.
/clear Clear conversation history. Start fresh while keeping CLAUDE.md.
/commit Stage and commit all Claude changes with an AI-generated commit message.
/undo Undo the last file change Claude made. Safe rollback.
/search <term> Search codebase for term. Claude uses results as context.
/diff Show all pending changes Claude has made but not yet saved.
/help List all available commands. Always up to date.
💡 Keyboard shortcut: ↑ in the Claude Code terminal recalls previous prompts — just like bash history.
