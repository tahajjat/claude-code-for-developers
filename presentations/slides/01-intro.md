    <div class="feature-card">

  <h1 class="slide-title">Claude Chat vs Claude Code</h1>
  <p class="slide-title-text">Understanding the difference — when to use which</p>

  <div class="r-hstack">

    <div class="col-left">
      <div class="compare-card chat-card">
        <h3>Claude Chat (claude.ai)</h3>
        <ul>
          <li>Web browser interface only</li>
          <li>Question → Answer model</li>
          <li>No access to your codebase</li>
          <li>Cannot edit files or run commands</li>
          <li>Great for: explaining concepts, writing docs, brainstorming</li>
          <li>Stateless between conversations</li>
        </ul>
      </div>
    </div>

    <div class="col-right">
      <div class="compare-card code-card">
        <h3>Claude Code (CLI + IDE)</h3>
        <ul>
          <li>Terminal + VS Code integrated</li>
          <li>Autonomous agent model</li>
          <li>Reads your entire project tree</li>
          <li>Edits files, runs tests, commits code</li>
          <li>Great for: building features, debugging, refactoring</li>
          <li>Maintains project context via CLAUDE.md</li>
        </ul>
      </div>
    </div>

  </div>

  <div class="tip-bar-t-slide-04">💡 WAMY Project: use Claude Code for all development tasks. Use Claude Chat for quick concept questions when not at your computer.</div>
